# Land Registration Management System - Frontend

This is the React frontend for the Land Registration Management System.

## Features
- React (Hooks)
- Material-UI (MUI)
- React Router
- Axios for API calls
- Context API for state management

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm start
   ```

## Project Structure
- `src/` - Main application code
- `public/` - Static files
